## 🎯 What is this?

A **community-driven showcase** where developers from around the world can:
- ✨ Display their profile card with photo, name, and role
- 🌟 Celebrate their first open source contribution
- 🤝 Connect with fellow developers
- 📚 Learn from real-world code examples

**Perfect for beginners** making their first PR!

---

## 🚀 Quick Start

### 1. Prepare Your Image
- **Format**: `.jpg` or `.png`
- **Size**: Square (1:1 ratio), under 500KB
- **Name**: `your-username.jpg` (e.g., `john-doe.jpg`)

### 2. Fork & Clone
```bash
git clone https://github.com/<your-username>/dev-card-showcase.git
cd dev-card-showcase
```

### 3. Add Your Card
1. Add your image to the `images/` folder
2. Open `index.html`
3. Find the comment `👇 CONTRIBUTORS: START COPYING FROM HERE 👇`
4. Copy the template and paste it at the bottom
5. Update with your details:

```html
<div class="card">
    <img src="images/your-image.jpg" alt="Your Name" class="card-img">
    <h2>Your Name</h2>
    <span class="role">Your Role</span>
    <p>"Your inspiring quote!"</p>
    <a href="https://github.com/yourusername" class="card-btn" target="_blank">GitHub</a>
</div>
```

### 4. Submit Your PR
```bash
git add .
git commit -m "Add [Your Name] to contributors"
git push origin main
```

Then create a Pull Request on GitHub!

## 🎨 Customization

Want to style your card uniquely? Add a custom class:

```html
<div class="card my-custom-style">
    <!-- your card content -->
</div>
```

Then add your styles to `style.css`:

```css
.my-custom-style {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
}
```

## 🐛 Troubleshooting

### Common Issues

**Image not loading?**
- Check file path and naming
- Ensure image is in `images/` folder
- Verify file format (.jpg or .png)

**Card not appearing?**
- Check HTML syntax
- Ensure you're adding after the template
- Verify all required fields are filled

**Theme not working?**
- Clear browser cache
- Check JavaScript console for errors
- Ensure localStorage is enabled
