# Compliment Mosaic Creator

A web app where each compliment becomes a colored tile in a collaborative mosaic, visualizing collective positivity.

## Features
- Send compliments and add to mosaic
- Each compliment is a colored tile
- Click tiles to view compliment messages
- Mosaic feed and dashboard
- Local storage for compliments

## Usage
Open index.html in your browser. Send compliments, view the mosaic, and celebrate collective positivity!
